render

render(target)

插件渲染函数，若定义此函数将会自动屏蔽html以及name属性，该函数需要返回一个html节点

@params target: 激活区dom

// test plugin
export default {
  id: 'demo',
  name: 'demo',
  html: '<div>demo</div>',
  display: true,
  render: (target) => {
    return '<div style="color:red;">demo-plugin</div>'
  }
}
